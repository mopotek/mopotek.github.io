Thalie, a cognitive disorder tracking software

Quality Assurance Testing • Heuristic analysis • User testing

2017 @ENSC, for Yumaneed & SimSoft3D

THALIE is a software created to help neuropsychologists track the evolution of their patient’s cognitive disorders like Alzheimer’s Disease. It allows the patients to go through examinations for standardized tests such as the Trail Making Test directly on a tablet assisted with voice recognition.

How could we improve the usability of a double sided tactile interface that was to be used for the elderly and healthcare professionals alike?

The objectives

  • Evaluate the usability of the current version of the software to improve it by communicating the necessary changes with the developers.
  • Create an engaging web-based user manual.

Which methods did I use?

1. Quality assurance testing

I performed walthroughs of every interface and possible behaviours to spot and document technical disfunctions prior to user testing, which were logged in a system to report issues to the SimSoft3D developpers.

2. Heuristic analysis

Using Bastien and Scapin's well documented 8 usability criteria, I performed a heuristic analysis of the interfaces and cognitive tests to formulate improvement recommendations to developers regarding cognitive load otpimizations, explicit controls for users, or error prevention.

3. User testing

We collected direct feedback from users by performing in-context user testing in retirement homes with senior citizens and 2 neuropsychologists.

The results were very useful to identify typography and elements that were hard to read for senior citizens.

4. Web tutorial development

We created short informative video tutorials for the neuropsychologists to easily access from a website. The tutorial was divided in many subsections to allow for quick access to the relevent information during usage.

Some results

Notable findings

  • A verified need: useful tool for the neuropsychologist that responds to their daily problems
  • Usability: font sizes and contrasts not adapted for senior citizens

Challenges

  • Finding the language to communicate with developers for the implementation of our solutions
  • Managing with deadlines not respected by other parties
